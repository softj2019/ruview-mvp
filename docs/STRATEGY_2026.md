## 2. 목표 시장 — 3단계 진입 전략

```
Phase A (2026 H1)         Phase B (2026 H2-2027)         Phase C (2027-2028)
─────────────────         ──────────────────────         ──────────────────
노인돌봄 시설             소규모 건물 관제               소방관제 + 스마트시티
독거노인 재택돌봄         다중이용업소 재실감지          화재 시 재실 정보 연동
                          ZEB 재실 기반 HVAC             디지털 트윈 통합
```

### Phase A: 노인돌봄 (즉시 진입 가능)

**시장 근거:**
- 독거노인 32.8%, 고독사 3,924명/년 (7.2%↑)
- 서울시 IoT 비대면 돌봄 1만+ 가구 설치, 135건 위험상황 감지
- 통합돌봄 예산 914억 (전국 183개 지자체)
- 노인맞춤돌봄 5,894억 (57.6만 명)

**ruView 적합성:**
| 요구사항 | ruView 대응 | 기존 솔루션 한계 |
|----------|------------|-----------------|
| 낙상감지 | CSI+카메라 융합 95%+ | 웨어러블 착용 거부율 높음 |
| 재실감지 | WiFi CSI 벽투과 | PIR 시야각 제한 |
| 프라이버시 | 영상 없음 | CCTV 사생활 침해 논란 |
| 비용 | ESP32 ~$10/대 | 레이더 $100+ |
| 바이탈 | 호흡 14-27, 심박 70-90 | 웨어러블 필요 |

**핵심 차별점: 요양시설 CCTV 의무화(2023.06) → 프라이버시 논란 → WiFi CSI가 비영상 대안**

**사업화 모델:**
- 디바이스: 50,000-100,000원 (ESP32-S3 3-6대/시설)
- 모니터링 구독: 10,000-30,000원/월
- B2G 조달: 통합돌봄 사업, 서울시 IoT 돌봄 확대

### Phase B: 소규모 건물 관제

**규제 기회:**
- 중대재해처벌법: 37개 스마트안전장비 지정, "AI 기반 인체감지시스템" 포함
- 스마트안전장비 보조금: **사업장당 최대 4,000만 원** (50인 미만)
- 2026년 예산 350억 원
- ZEB 의무화 확대: 민간 주거단지(2025.06~), 점유 기반 HVAC 필수

**비용 비교:**
| 항목 | PIR | WiFi CSI | 카메라 |
|------|-----|----------|--------|
| 대당 비용 | $20-30 | **$10** | $100-500 |
| 커버리지 | 1방 (시야각) | **1,500sqft 3대** (벽투과) | 시야각 |
| 설치 | 배선 필요 | **기존 WiFi 활용** | 배선+스토리지 |
| 프라이버시 | 비침습 | **비침습** | 침습적 |

### Phase C: 소방관제 + 스마트시티

**소방 연동:**
- NFSC 203 자동화재탐지설비 + WiFi CSI 재실 데이터 = 대피 우선순위
- 화재 시 구조대원에게 재실 여부 실시간 전달
- 벽투과 감지로 연기 속 생존자 탐색
- KFI 인증 선점 (WiFi CSI 기반 최초)

**스마트시티 통합:**
- 부산 에코델타시티 (5.6조 원 투자, 2026 완공)
- 세종 5-1 생활권 디지털트윈
- 스마트빌딩 디지털트윈에 점유 데이터 피드

## 3. 기술 고도화 로드맵

### 3.1 정확도 벤치마크 (목표)

| 지표 | 현재 (ruView) | 목표 | SOTA |
|------|:------------:|:----:|:----:|
| 재실감지 | 90%+ | **98%+** | 100% (CNN) |
| 낙상감지 | 80% (ML프레임워크) | **95%+** | 95.5% (CNN-LSTM) |
| 호흡수 | MAD ~3 bpm | **MAD <1.5** | 0.13 brpm |
| 심박수 | 추출 중 | **MAD <1.0** | 0.08 BPM (Pulse-Fi) |
| 다중인원 | 1-5명 추정 | **±1 정확도** | 단일공간 제한 |
| 자세분류 | 4종 | **6종 + 제스처** | 99.9% (PA-CSI) |

### 3.2 기술 스택 진화

```
현재 (v0.1)                    단기 목표 (v0.2)                장기 목표 (v1.0)
───────────                    ─────────────                   ──────────────
ESP32-S3 CSI 추출              + TFLite Micro 엣지 추론        + 802.11bf 정합
Python signal-adapter          + Rust 고성능 파이프라인          + WASM 브라우저 처리
규칙 기반 감지                  + CNN-LSTM 딥러닝 모델           + 연합학습 프라이버시
수동 캘리브레이션               + Transfer Learning 자동적응      + Zero-shot 배포
```

### 3.3 핵심 기술 과제

| 과제 | 내용 | 참조 |
|------|------|------|
| **엣지 AI** | ESP32-S3 TFLite Micro + ESP-NN, 232ms 추론 | esp-tflite-micro |
| **WASM 신호처리** | ruview-wasm crate → 브라우저 FFT/필터링 | Web Audio + WASM |
| **전이학습** | 환경 독립적 모델, 설치 즉시 동작 | SSL + 메타러닝 |
| **연합학습** | 원시 CSI 미전송, 모델만 업데이트 | FedWiLoc (61.9%↑) |
| **802.11bf 정합** | CSI 측정 포맷 표준 준수 | IEEE Std 802.11bf-2025 |
| **수면 모니터링** | 호흡+심박 기반 4단계 수면분류 | WiFi CSI Sleep 연구 |
| **HRV 분석** | SDNN, RMSSD, PNN50 심박변이도 | 스트레스/건강지표 |

## 4. 인허가 & 인증 전략

### 4.1 식약처 경로 (2단계)

```
Step 1: 디지털의료-건강지원기기 (웰니스)
   ├── 건강 유지/향상 목적
   ├── 완화된 규제 적용
   ├── 재실감지, 수면모니터링, 활동량
   └── 시장 진입 + 임상 데이터 축적

Step 2: 디지털의료기기 Class II
   ├── 질병 진단/예후 관찰
   ├── 낙상감지, 바이탈 이상 알림
   ├── 임상시험 필요
   └── FDA 510(k) 선례: Xandar Kardian XK300
```

### 4.2 소방 인증 (KFI)

- 한국소방산업기술원 형식승인 대상 32개 품목
- WiFi CSI 재실감지 = **기존 카테고리에 없음** → 신규 품목 선제 협의
- 자동화재탐지설비 보조장치로 포지셔닝

### 4.3 프라이버시 (개인정보보호법)

| 사용 형태 | PIPA 분류 | 조치 |
|-----------|----------|------|
| 익명 재실감지 (유/무) | **비개인정보** | 추가 조치 불요 |
| 활동 패턴 모니터링 | 회색지대 | 동의 획득 권장 |
| 개인 식별 (걸음걸이) | **민감정보** | 명시적 동의 필수 |

**ruView 전략: 엣지 처리 + 익명 이진 상태 출력 → 개인정보 비해당**

# Frontend Agent

## Role
React shell, 플로어 차트, Zone/Coverage/Event panel, 운영 상태 UI

## Tools
- React / Vite / TypeScript
- Tailwind CSS / shadcn/ui
- Zustand
- SVG / Konva
- Recharts

## Inputs
- WebSocket event stream
- Supabase realtime data
- Zone/device configuration

## Outputs
- Dashboard page with KPI cards
- 2D floor view with zones and devices
- Alert panel
- Device list
- Signal charts

## Completion Criteria
- 감지 범위와 감지 클래스가 화면에서 즉시 확인 가능
- 실시간 이벤트가 1초 이내에 UI 반영
- 반응형 레이아웃 동작

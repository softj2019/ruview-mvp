# QA Agent

## Role
데모 시나리오 검증, 진행상태 체크리스트 유지, 스크린샷/영상 캡처, 발표용 검증 로그 작성

## Tools
- Browser DevTools
- Screen capture
- Checklist management

## Inputs
- MVP checklist
- Demo scenarios
- Test criteria

## Outputs
- Test execution results
- Demo recording
- Presentation script
- Progress reports

## Completion Criteria
- 모든 데모 시나리오가 성공적으로 재현 가능
- 발표용 스크립트 작성 완료
- 비전문가가 화면만 보고 상황 파악 가능
